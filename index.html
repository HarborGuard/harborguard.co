Harbor Guard is a comprehensive container security scanning platform that provides an intuitive web interface for managing and visualizing security assessments of Docker images.


Get started with:

    docker run -p 3000:3000 ghcr.io/harborguard/harborguard:latest
View on GitHub

Multi-Tool Security Scanning

Harbor Guard integrates multiple industry-standard security scanning tools including Trivy, Grype, Syft, Dockle, OSV Scanner, and Dive into one unified platform.

  • Vulnerability detection and classification
  • Interactive vulnerability charts
  • SBOM generation

Secure your container images and understand what vulnerabilities exist in your Docker environments.

Advanced Visualization & Reporting

Interactive scatter plots, historical tracking, and comprehensive report export capabilities with real-time progress monitoring.

  • Real-time Scan Progress
  • Report Export (JSON/ZIP)
  • Historical Scan Comparison

Secure your containers with enterprise-ready features and comprehensive vulnerability management.
